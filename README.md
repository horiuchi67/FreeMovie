# <アプリ名>
FreeMovie

## サイト概要
映画の感想の投稿、画像投稿ができ、コメント機能、ログイン機能、フォロー機能、非同期通信、メール配信、地図表示、通知があります。
S3を使用します。

### サイトテーマ
感想投稿SNS

### テーマを選んだ理由
私は映画を見るのが趣味で、同じ趣味を持つ人たち向けのSNSを作りたかったから。

### ターゲットユーザ
映画を見た人や映画に関心がある人。

### 主な利用シーン
映画を見終わった後や、どの映画を見ようか迷っているとき、時間つぶしの時に利用します。
